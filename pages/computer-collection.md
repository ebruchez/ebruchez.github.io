# Computer collection

## Introduction

In addition to typewriters, I have been collecting vintage computers, teletypes, terminals, and a few other accessories for a while.
Here is the current collection.

## Vintage terminal collection

As of March 2026, I count 8 teletypes/terminals, if I include what might end up as parts machines. The earliest
terminals are electromechanical, while the later ones are fully electronic (and sometimes called "glass teletypes").

## Vintage computer reproductions collection

In addition to originals, I also have a few reproductions. I bought the PiDP-11, as well as the Altair-Duino, as kits.
But I worked hard to make the Apple-1 reproduction, in spite of standing on the shoulders of giants.

<div class="tw-table">

<figure>
    <img src="/assets/pages/computer-vintage/pidp-11.webp"/>
    <figcaption>
        <a href="https://obsolescence.wixsite.com/obsolescence/pidp-11">PiDP-11</a>. This reproduction, based on a
        Raspberry Pi and the SimH emulator, can run multiple of the original operating systems running on the original
        PDP-11. This picture shows it connected to a real Televideo terminal and running Unix.
    </figcaption>
</figure>
<figure>
    <img src="/assets/pages/computer-vintage/apple-1.webp"/>
    <figcaption>
        Apple-1. I built this reproduction of an Apple 1 between 2024 and 2025. It is fully functional, including the
        tape drive. For more, see my <a href="https://blog.bruchez.name/posts/apple-1-reproduction-part-1-components/">series of blog posts</a>. 
        The only missing part is a case, which I'd like to be at least partially transparent so that the main board can
        be seen.
    </figcaption>
</figure>
<figure>
    <img src="/assets/pages/computer-vintage/altair-duino.webp"/>
    <figcaption>
        <a href="https://adwaterandstir.com/">Altair-Duino</a>.
        This reproduction, based on an Arduino Due board, emulates the front panel of an Altair 8800. Here, the
        Altair is connected to the Televideo terminal and is running WordStar on CP/M! The PiDP-11 is seen on top of
        the Altair-Duino.
    </figcaption>
</figure>
